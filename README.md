## Описание

**deadliner** - простая консольная утилита, которая используется для управления дедлайнами.
Все записи выстрены по времени, от ближайшей к будущим.
```
09.11.2025 10:30 Запись 1
09.11.2025 14:00 Запись 2
31.12.2025 23:59 Запись 3
```
Все записи хранятся в файле, а изменения записываются только при выходе из программы.

## Установка
Программа написана для Linux и пока работает только с ним.

1. Если не установлены **git** и **python** сначала установите их.

2. Скачайте репозиторий командой
```
git clone git@github.com:gh-Neniy/deadliner.git
```

3. Из полученной папки `/deadliner` запустите установочный скрипт
```
python3 install.py
```

## Команды
Чтобы запустить программу:
```
deadliner
```

Чтобы сохранить изменения и выйти:
```
exit
```

Ближайшая запись:
```
top
```

Все записи:
```
all
```

Добавить новую запись:
```
add [day].[month].[year] [hour]:[minute] [text]

# Например:
# 19.10.2025 11:25 Новая запись
```
```
add [day].[month] _ [text]

# Можно не писать год и время:
# 22.10 _ Новая запись

# Тогда запись сохранится с добавлением текущего года и c 23:59:
# 22.10.2025 23:59 Новая запись
```

Удалить ближайшую запись:
```
pop
```

Удалить конкретную запись:
```
# Работают сокращения, как и в add:
remove 22.10 _ Новая запись
```
```
# Вместо remove можно использовать rm для удобства:
rm 22.10 _ Новая запись
```
## Флаги
Можно использовать флаг `--top`, чтобы вывести ближайшую запись, не запуская интерактивную часть программы:
```
deadliner --top
```

И так же с `--all`:
```
deadliner --all
```
